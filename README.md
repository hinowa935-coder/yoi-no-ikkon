# 宵ノごはん

日本酒と家庭料理のペアリングを、料理・気分・産地・味わい・夜の言葉から探せるNext.jsアプリです。

## ローカルで確認

```bash
npm install
npm run dev
```

## 公開用に書き出す

```bash
npm run build
```

ビルド後、`out/` フォルダに公開用ファイルが作られます。

## 公開先

おすすめはVercelです。GitHubにこのフォルダをアップして、Vercelでそのリポジトリを選ぶと公開できます。

Vercelを使わない場合も、`out/` フォルダを静的サイトとして公開できるサービスにアップできます。
